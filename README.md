# 🌦 ClimateX Weather Platform

A modern full-stack MERN weather application with real-time forecasts, AQI analytics, authentication, charts, favorites, and responsive dashboard UI.

---

# 🚀 Features

## 🌍 Weather Features
- Real-time weather data
- 5-day forecast
- Hourly forecast
- Air Quality Index (AQI)
- Current location weather
- Search weather by city

## 🔐 Authentication
- User Signup
- User Login
- JWT Authentication
- MongoDB user storage

## 🎨 UI Features
- Modern responsive dashboard
- Dark mode
- Animated cards
- Weather charts
- Mobile friendly design
- Favorites section

---

# 🛠 Tech Stack

## Frontend
- React.js
- Tailwind CSS
- Axios
- Framer Motion
- Recharts

## Backend

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/climatex-weather-platform.git
```

---

## 2️⃣ Frontend Setup

```bash
cd client
npm install
npm start
```

Frontend runs on:

```bash
http://localhost:3000
```

---

## 3️⃣ Backend Setup

```bash
cd server
npm install
node index.js
```

Backend runs on:

```bash
http://localhost:5000
```

---

# 🔑 Environment Variables

Create `.env` inside `server` folder.

```env
MONGO_URI=YOUR_MONGODB_URI
JWT_SECRET=climatexsecret
```
